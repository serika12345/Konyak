## Highlights

- Added a complete Konyak Wine runtime update path in the Flutter UI. When a
  managed macOS Wine runtime update is available, Konyak now shows a localized
  confirmation prompt instead of only relying on startup notification text.
- Connected confirmed runtime updates to the existing JSON CLI contract:
  `install-runtime-update <runtime-id> --json`. The app refreshes runtime state
  after the install path finishes so the Settings runtime list reflects the new
  version.
- Extended the manual `Check for Updates...` command so it checks Konyak app
  releases first and then checks installed managed runtime releases, matching
  the startup update behavior.
- Added macOS Wine runtime release-feed support for newly published runtime
  versions. This allows an installed Konyak runtime to detect
  `crossover-26.1.0-konyak.2`, the current cabextract-capable runtime release,
  as an installable update from the runtime Releases feed.
- Added a development and CI override for the macOS runtime release feed through
  `KONYAK_MACOS_WINE_VERSION_URL`, making runtime update checks testable
  without mutating the production Releases page.
- Updated the macOS runtime submodule pointer to the released
  `crossover-26.1.0-konyak.2` line. GPTK/D3DMetal payloads remain
  user-provided; Konyak still does not bundle or redistribute Apple GPTK
  binaries.

## Fixes and Maintenance

- Split the runtime update UI logic out of the broader home loader runtime
  module so the update flow stays testable and under the repository governance
  file-size limit.
- Hardened Pages deployment reruns by adding a small concurrency guard, reducing
  repeat failures when release and documentation pushes happen close together.
- Added CLI contract coverage for runtime update checks that compare installed
  Wine component versions against runtime release tags, report newly added macOS
  runtime release versions, and reject runtime releases without source
  manifests.
- Added Flutter widget coverage and a golden for the Konyak Wine update
  confirmation prompt.
