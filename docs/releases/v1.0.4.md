## Highlights

- Added static graphics backend hints for Windows programs so Konyak can inspect
  a selected executable and suggest likely graphics paths before launch.
- Added a maintained VSCode and CLI release flow with version selection,
  checked-in Markdown release notes, release candidate gates, rollback on
  failure, annotated tag creation, push, and publish workflow dispatch.
- GitHub Releases now include the checked-in release notes for the tag before
  the generated SHA-256 checksum section.

## Verification

- Local release preparation runs `just release-candidate-gates` before creating
  the release commit and tag. On macOS this includes `just verify`, the macOS
  release build, packaged runtime extraction, DMG layout, Finder integration,
  packaged app CLI bridge, and app update handoff smokes.
- The GitHub Actions publish workflow verifies repository checks, the Linux
  AppImage artifact, and the macOS DMG artifact before publishing release
  assets.
