Platform aware

macOS first. Linux next.

Konyak starts with arm64 macOS runtime support while keeping x86_64 Linux behavior isolated behind platform services.

macOS

Konyak-managed Wine runtime support for arm64 hosts, with platform-specific settings isolated from Linux.

Linux

Wine and Proton runtimes for x86_64, DXVK for D3D8-D3D11, and vkd3d-proton for D3D12.
